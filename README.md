# abidurrahman2025.github.io

This repository is a GitHub Pages user site for `abidurrahman2025`. Drop the static site files at the repository root on the `main` branch and the site will be available at:

https://abidurrahman2025.github.io

## Included
- `index.html` — single-page responsive portfolio template
- `assets/styles.css` — styles
- `.nojekyll` — ensure GitHub Pages serves files directly

## How to publish locally / push
1. Clone (if you haven't already):
   ```
   git clone https://github.com/abidurrahman2025/abidurrahman2025.github.io.git
   cd abidurrahman2025.github.io
   ```
2. Add or replace the files, then:
   ```
   git add .
   git commit -m "Add static site scaffold"
   git push origin main
   ```
3. Visit: `https://abidurrahman2025.github.io` (it may take a minute to appear)

## Customize
- Replace copy, links, and email in `index.html`.
- Replace project cards with real project links.
- Replace the favicon data URI or add `favicon.ico` in repo root.
